흙 패치 이미지 Auto Tile 등록 예시

이 가이드는 올려준 이미지처럼 풀 배경 위에 흙 패치가 있는 3x3 타일셋을 현재 프로젝트의 Tiles By Mask에 어떻게 넣을지 설명합니다.

기준은 흙 타일끼리 연결되는 Auto Tile Set입니다. 즉 Tile Id는 예를 들어 DirtA로 두고, 주변에 같은 DirtA가 있으면 연결된 것으로 계산합니다.

1. 이미지 자르기

  1. 이미지를 Unity에 넣습니다. 예: Assets/Tiles/Dirt/dirt_patch.png
  2. Inspector에서 Texture Type을 Sprite (2D and UI)로 설정합니다.
  3. Sprite Mode를 Multiple로 바꿉니다.
  4. Sprite Editor에서 이미지를 3 x 3으로 자릅니다.
  5. 잘린 9개 스프라이트로 Tile asset을 만듭니다.

위 그림의 숫자가 Tiles By Mask 배열 인덱스입니다. 예를 들어 이미지의 가운데 타일은 모든 방향이 흙으로 연결되므로 Tiles By Mask[255]에 넣습니다.

위쪽 가운데 조각은 왼쪽, 오른쪽, 아래, 왼쪽 아래, 오른쪽 아래가 흙이므로 64 + 4 + 16 + 32 + 8 = 124입니다.

4. Auto Tile Set 설정 순서

  1. Create > Tools > World > Auto Tile Set을 만듭니다.
  2. Tile Id를 DirtA로 설정합니다.
  3. Fallback Tile에는 가운데 흙 타일이나 가장 무난한 기본 타일을 넣습니다.
  4. 위 표에 맞춰 Tiles By Mask의 해당 인덱스에 9개 Tile asset을 넣습니다.
  5. Create > Tools > World > Auto Tile Palette를 만들고 이 Auto Tile Set을 등록합니다.
  6. 씬의 SampleSceneBootstrap에 있는 Auto Tile Palette 필드에 팔레트를 연결합니다.

5. 현재 프로젝트의 8방향 값

128 1 2 64 X 4 32 16 8
이 이미지만으로는 큰 흙 지형의 외곽 8방향과 내부 중앙은 처리할 수 있지만, 한 칸짜리 섬, 얇은 길, T자, 십자, 안쪽 코너 같은 모든 경우를 완전히 처리하지는 못합니다. 없는 mask는 Fallback Tile로 표시됩니다.
