TileMap 타일 세팅 가이드

이 문서는 현재 프로젝트의 직접 구현된 8방향 bitmask auto tile 렌더러를 사용하기 위해 Unity에서 준비해야 할 데이터 작업을 정리한 가이드입니다.

1. 준비할 에셋

에셋 역할 예시 경로
Tile Asset 실제로 화면에 그려질 Unity Tile 또는 TileBase Assets/Tiles/Water/Water_255.asset
Auto Tile Set 하나의 TileId가 mask별로 어떤 Tile을 쓸지 정의 Assets/Data/AutoTiles/WaterShallowAutoTileSet.asset
Auto Tile Palette 여러 Auto Tile Set을 렌더러에 연결하는 팔레트 Assets/Data/AutoTiles/AutoTilePalette.asset

2. Tile Asset 만들기

  1. 스프라이트 시트를 Assets/Tiles 아래에 넣습니다.
  2. Sprite Mode를 Multiple로 설정하고 Sprite Editor에서 타일 단위로 자릅니다.
  3. 잘린 스프라이트를 사용해 Unity Tile asset을 만듭니다.
  4. 물, 벽, 길처럼 연결 모양이 필요한 타일은 mask별 Tile asset을 준비합니다.
처음부터 256개를 모두 채울 필요는 없습니다. Fallback Tile을 지정한 뒤, 자주 보이는 mask부터 단계적으로 채우면 됩니다.

3. Auto Tile Set 만들기

  1. Project 창에서 우클릭합니다.
  2. Create > Tools > World > Auto Tile Set을 선택합니다.
  3. Tile Id에 대상 타일을 고릅니다. 예: WaterShallow.
  4. Fallback Tile에 기본 표시용 Tile을 넣습니다.
  5. Tiles By Mask 배열의 인덱스에 mask별 Tile을 넣습니다.

4. 8방향 Bitmask 규칙

현재 타일 X 기준으로 주변 8칸을 검사합니다.

128 1 2 64 X 4 32 16 8

같은 auto tile 그룹으로 연결되는 방향의 값을 더한 숫자가 Tiles By Mask의 인덱스입니다. 

위 + 오른쪽 + 아래 + 왼쪽
= 1 + 4 + 16 + 64
= 85

대각선은 양쪽 직교 방향이 모두 연결된 경우에만 인정됩니다. 예를 들어 오른쪽 위 대각선은 위, 오른쪽, 오른쪽 위가 모두 연결될 때만 2가 더해집니다.

5. Connects To 설정

서로 다른 TileId가 같은 덩어리처럼 이어져야 한다면 Connects To에 추가합니다.

Auto Tile Set Connects To 예시 의미
WaterShallow WaterDeep, SwampShallow 얕은 물이 깊은 물/늪과 이어져 보임
RoadA 없음 같은 길 타일끼리만 이어짐
OldWall RockLarge 벽과 큰 바위가 같은 장애물 덩어리처럼 이어짐

6. Auto Tile Palette 만들기

  1. Create > Tools > World > Auto Tile Palette를 선택합니다.
  2. Tile Sets 배열에 만든 Auto Tile Set들을 추가합니다.
  3. 예: WaterShallowAutoTileSet, RoadAAutoTileSet, OldWallAutoTileSet.

7. 씬에 연결하기

  1. 씬에서 SampleSceneBootstrap 컴포넌트가 붙은 오브젝트를 선택합니다.
  2. Auto Tile Palette 필드에 만든 팔레트 에셋을 드래그합니다.
  3. Play를 실행해서 타일 연결 모양을 확인합니다.

8. 확인할 것

  • 타일이 분홍색으로 나오면 해당 mask의 Tile asset이 없고 fallback도 없는 상태입니다.
  • 모서리가 이상하면 대각선 mask에 들어간 Tile asset을 확인합니다.
  • 원하는 타일끼리 이어지지 않으면 Connects To 설정을 확인합니다.
  • Auto Tile Set을 연결하지 않은 TileId는 기존 디버그 색상 타일로 표시됩니다.
